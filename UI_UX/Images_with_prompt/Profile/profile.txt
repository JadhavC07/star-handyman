Mobile app profile screen UI for a handyman service provider.

Use consistent design system:
- Primary color: #FF7A00 (orange)
- Secondary color: #1E2A78 (dark blue)
- Clean white background
- Rounded mobile layout
- Modern, minimal UI

Layout:

Top section:
- Centered profile avatar (circular, large)
- Name: "John Carter"
- Rating: 4.8 ★ with review count (e.g., 120 reviews)
- Small verification badge (verified icon)

Info section (card style):
- Skills: Plumbing, Electrical, Carpentry (tag-style chips)
- Experience: 5+ years
- Location (optional)

Portfolio preview (optional but recommended):
- Horizontal scroll images of past work

Main menu sections (list with icons):
- Edit Profile
- Documents / KYC verification
- Payment Settings
- Notifications (optional)
- Help & Support (optional)

Each item:
- Icon (left)
- Label text
- Chevron arrow (right)

Bottom section:
- Logout button (outlined or subtle)

Style:
- Rounded cards (24px radius)
- Soft shadows, subtle elevation
- Pill-style chips for skills
- Clean typography hierarchy
- 8pt spacing system

Interaction hints:
- Touchable list items
- Highlight on press (subtle)

Lighting:
- Bright, soft UI lighting
- No blur, no noise

Output:
- Ultra sharp, 4K resolution
- Pixel-perfect mobile UI
- App Store quality design

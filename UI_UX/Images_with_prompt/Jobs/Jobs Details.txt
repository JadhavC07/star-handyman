Mobile app job detail screen UI for a handyman service provider.

Use consistent design system:
- Primary color: #FF7A00 (orange)
- Secondary color: #1E2A78 (dark blue)
- White background
- Rounded mobile layout
- Clean, modern UI

Layout:

Top section:
- Header with back button + title "Job Details"

Customer section:
- Customer name (e.g., Sarah Johnson)
- Rating (4.7 ★)
- Profile avatar (small circle)

Job overview card:
- Job title: "Fix leaking kitchen sink"
- Category: Plumbing
- Description: short paragraph explaining the issue
- Posted time: "2 hours ago"

Location section:
- Address + distance (e.g., 3.2 km away)
- Small map preview or location icon

Pricing section:
- Budget range (e.g., $50–$80)
- Payment type (fixed / hourly)
- Highlight price in bold

Media section:
- Images uploaded by customer (horizontal scroll)
- Optional: 2–3 thumbnails

Schedule section:
- Preferred date and time

Action section (bottom fixed):
- Primary button: "Accept Job" (orange)
- Secondary button: "Send Quote" or "Message" (dark blue or outlined)

Style:
- Cards with rounded corners (24px)
- Soft shadows, clear separation between sections
- Clean typography hierarchy
- 8pt spacing system

UX details:
- Sticky bottom action bar
- Clear visual hierarchy (problem → location → money → action)
- No clutter, high readability

Lighting:
- Bright, soft UI lighting
- No blur, no noise

Output:
- Ultra sharp, 4K resolution
- Pixel-perfect mobile UI
- App Store quality design
